업그레이드 시스템

| 항목       | Manager?  | System? | 이유                                     |
| -------- | --------- | ------- | -------------------------------------- |
| Input    | 매니저       | ❌ 시스템   | 입력 상태는 전역적 상태 → Manager                |
| Build    | ❌ 매니저     | 시스템     | 건물 설치는 로직 → System                     |
| Grid     | 매니저 + 시스템 | 둘 다     | Grid 데이터 저장 → Manager / 좌표 계산 → System |
| Event    | 매니저       | ❌       | 이벤트를 저장·발동만 하기 때문에 Manager             |
| UI       | ❌         | 시스템     | UI를 변경하는 행위 중심                         |
| Resource | 매니저       | ❌       | 자원 수치를 들고 있고 변경 저장                     |
| Citizen  | 매니저       | ❌       | 모든 시민 목록, 직업 등을 저장                     |
| Upgrade  | ❌         | 시스템     | 업그레이드 판단 & 실행 로직                       |
