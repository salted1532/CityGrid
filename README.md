# 🏙️ 도시 건설 경영 시뮬레이션

## 장르
- 건설
- 경영
- 시뮬레이션

---

## 핵심 시스템
- 시민 세금을 통해 건설 및 도시 확장
- 세금은 시민 **계층별로 개별 적용 가능**
- 세율, 시설 상태에 따라 **인구 증가 / 감소 / 불만도 변화**

---

## 주요 자원
| 자원 | 설명 |
|---|---|
| 돈 | 건설, 유지비, 업그레이드 |
| 전기 | 건물 작동에 필요 |
| 물 | 시민 생활에 필요 |

---

## 도시 지표
| 수치 | 효과 |
|---|---|
| 물 공급 | 부족 시 인구 성장이 멈춤 |
| 쓰레기 처리 | 부족 시 불만 증가 |
| 치안 | 낮을수록 범죄/소요 발생 |
| 화재 발생률 | 높을수록 발화 위험 |
| 질병 발생률 | 병원 미배치 시 증가 |
| 실업률 | 실업 증가 → 치안 악화 + 세금 감소 |
| 교육률 | 공장 업그레이드 및 화재 감소 |
| 도시 인기도 | 이주율 결정 |

---

## 불만 시스템
- 인구가 일정 수에 도달할 때마다 **요구 이벤트 발생**
- 요구 미충족 시 **불만 수치 증가 → 인구 증가 속도 저하**

# 건설 시스템
- 3D 환경 + 2D 그리드 기반
- 건물은 반드시 **도로 인접** 시 활성화

---

## 시민 & 자동차 시스템
- 시민과 차량은 도로를 따라 이동
- **목표 지점으로 이동 → 목표 도달 후 제거 → 새로운 목적지로 재스폰** 반복
- 성능 최적화를 위한 **시뮬레이션 방식 경로 이동** 구조

---

## 땅값 시스템
- **땅값은 주거 건물 업그레이드 조건에 직접 영향**
- 특정 수치/범위에 따라 주거 건물의 **계층 변화 및 업그레이드/다운그레이드** 발생
- 공원, 병원, 은행 등 **고급 건물 인접 시 땅값 상승**
- 쓰레기장, 발전소 등 **오염/부정 이미지 건물 인접 시 땅값 하락**

---

## 상수도 (물) 시스템
- 모든 시민/건물이 **물 소비량**을 가짐
- **인구 증가 → 물 공급 요구 증가**
- 물 공급량이 부족할 경우:
  - **인구 성장 정지**
  - 불만 수치 상승 가능

---

## 쓰레기 처리 시스템
- 도시 규모가 커질수록 **쓰레기 배출량 증가**
- 쓰레기장 / 소각장 / 재활용 시설 배치 필요
- 쓰레기 처리량 부족 시:
  - 시민 **불만 증가**
  - 도시 위생/인기도 저하

---

## 질병 시스템
- 시민은 일정 확률로 **질병에 감염**
- 감염 시:
  - 일정 시간 경과 후 **사망 또는 회복 판정**
- 병원 범위 내 주거 지역은:
  - **회복 확률 증가**
  - 회복 실패 시 **재시도 사이클 반복**

---

## 치안 시스템
| 요인 | 영향 |
|---|---|
| 빈곤층 증가 | 치안 하락 → 범죄 증가 |
| 실업률 증가 | 치안 하락 → 범죄 증가 |
| 치안 수치 하락 | 도시 인기도 하락 → 이주율 감소 |

- 치안 확보를 위해 **경찰서/감옥 배치 필요**

---

## 화재 시스템
- 건물은 일정 확률로 **화재 발생**
- 발생 시:
  - 소방서에서 **소방차 출동 → 현장 도착 후 진압**
- 진압 실패(지정 시간 경과) 시:
  - 건물 **소멸**
- 화재 발생률은 **시민 교육 수준이 높을수록 감소**

---

## 일자리 & 세금 시스템
- 시민 계층별로 **선호하는 일자리 유형이 존재**
- 실업자가 많을수록:
  - 치안 악화
  - **세금 수입 감소**
- 모든 시민이 일자리를 가질 경우:
  - **세금 100% 효율 확보**
